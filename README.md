# CPU Scheduler Simulator

A modern GUI-based CPU Scheduling Simulator built using Python, Tkinter, and Matplotlib.

This project visualizes how different CPU scheduling algorithms work using:
- Interactive Gantt Charts
- Performance Metrics
- Process Tables
- Algorithm Comparisons

---

# Features

## Supported Scheduling Algorithms
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- SRTF (Shortest Remaining Time First)
- Round Robin
- Priority Scheduling
- Compare All Algorithms Mode

---

# GUI Features

- Dark modern UI
- Interactive process management
- Dynamic Gantt chart visualization
- Animated performance metrics
- CPU utilization calculation
- Throughput analysis
- Algorithm comparison dashboard
- Demo process loader
